# Обнаруженные дефекты в Утилитах для обработки и анализа строк
-------------------------------------------------------------------------------------------------------------------------------------------
1.
Краткое описание		-	Функция "contains" принимает в качестве аргумента для поиска вместо одиночного символа строку
Приложение		-	StringUtils.py
Компонент приложения	-	Функция "contains", определяющая наличие указанного символа в исходной строке
Версия приложения	-	Не указана
Серьезность бага		-	Средняя
Приоритет бага		-	Средний (Выставляет менеджер проекта)
Окружение		-	Windows 10 22H2, Python 3.13.0 64-bit
Шаги воспроизведения	- 	В качестве второго входного параметра при вызове функции "contains" использовать строку из двух и более символов
Фактический результат	- 	Функция принимает вторым параметром строку из более чем одного символа и осуществляет ее поиск в исходной строке
Ожидаемый результат	-	Выдача сообщения о некорректности входных данных.
